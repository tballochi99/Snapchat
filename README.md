## Installation

Pour installer et utiliser Snapchat :

1. Clonez ce dépôt : `git clone git@github.com:tballochi99/Snapchat.git`
2. Naviguez vers le répertoire du projet : `cd snapchat`
3. Installez les dépendances : `npm install`

## Utilisation

```bash
$ npm start
